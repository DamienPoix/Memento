↑ HTML ↓

Les listes

La balise "ol"permet de créer des listes ordonnée

la balise "ul"permet de créer des listes non ordonnée

pour créer une liste nous avons besoin d'une balise "ul"ou"ol" ainsi qu"une balise"li"

"li" est un item dans une liste

Texte

la balise "em"permet de mettre en italic en html et de donner de l'importance

La balise "mark" permet de mettre en gras en html et donner de l'importance

Les balises "h1""2""h..""h6"sont les différentes balises de titre du plus important au moins important
Il est impossible de mettre une balise h6 sans la balise qui la précéde (h1/h2/h3...)

La balise "p" permet de faire un paragraphe

les balises auto fermante

la balise auto fermante "br/" permet de sauter des lignes dans un paragraphe

la balise auto fermante "img/" permet a l'aide d'une source d'introduire une image sur la page "

La balise auto fermante "input" permet de définir un champ de formulaire (texte,buton,ect...)

Les tableaux

La balise "thead" permet de créer des cellules d'en tête dans un tableau

La balise "tbody" permet de créer des cellules corp dans un tableau

La balise "tfoot" permet de créer les cellules pied du tableau

La balise "tr" est une ligne d'un tableau de donnés elle comprend la balise "th" ou "td" pour afficher les informations

On peut ajouter "colspan="[nombre de cellules]" aux balises "td" et "th" pour fusionner un certain nombre de cellules

On utilise la balise "caption" a la suite de la formule table pour donner un titre au tableau

Les formulaires

L balise "fieldset"permet d'encadrer le formulaire ce trouvant a l'intérieure de cette balise

L balise "legend" permet de nommer un fieldset

La balise "textarea"permet de faire une zone de texte multi ligne

L balise "label" permet de féfinir un label a une balise"input"

    On peut ajouter a input les atribute type suivant:
  • type="text"
  • type="radio"
  • type="button"
  • type="mail"
  • type="submit"

↑ CSS ↓

Pour modifier la hauteur d'un block sur css on utilise: height:[taille];

Pour modifier la largeur d'un block sur CSS on utilise: width:[taille];

Pour modifierles bordure d'un block on utilise: border:[épaisseur] solid [couleur]

Pour changer la police d'un texte on utilise: font-family:[nom de la police];

Pour changer la couleur de la police on utilise: color:[couleur];

Pour changer la taille de la police on utilise: font-size:[Taille];

Pour aligner un texte on utilise: text-align:[center,left,right];

Pour changer la couleur du fond on utilise: background-color[couleur];

Pour changer la marge global exterieur on utilise: margin:[Taille];

Pour changer la marge exterieur on utilise: margin-[top,bottom,left,right]:[Taille];

Pour changer la marge global intérieur on utilise: padding[Taille];

Pour changer la marge intérieur on utilise: padding-[top,bottom,right,left]:[taille];

Pour faire un dégradé a l'aide de css on utilise background-image:linear-gradient(couleur ...);

Pour utiliser une image avec css on utilise background-image:url(URL de l'image);

Pour arrondir les angles des bordure on utilise border-[top/bottom]-[left/right]-radius: [taille 1] [taille 2];

pour spécifié ou un block doit etre placé on utilise : float [left/right/none]
